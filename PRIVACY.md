# Privacy Policy

Effective date: 2026-07-04

Temporary Chat Auto for ChatGPT & Claude is an unofficial Chrome extension that helps open new ChatGPT chats with Temporary Chat enabled and new Claude chats in Incognito mode.

## Data Collection

This extension does not collect, transmit, sell, or share personal data.

The extension does not send ChatGPT or Claude conversation content, account information, prompts, responses, browsing history, analytics, or identifiers to any external server.

## Extension Settings

The extension stores only user-selected on/off preferences: whether automatic Temporary Chat is enabled for ChatGPT and whether automatic Incognito chat is enabled for Claude. These preferences are saved with Chrome's extension sync storage (`chrome.storage.sync`). If you are signed in to Chrome with sync enabled, Chrome may store them in your Google Account and sync them across your browsers; this is handled entirely by Chrome. The developer never receives or can access these settings, and no other data is stored.

## Host Access

The extension runs only on:

- `https://chatgpt.com/*`
- `https://chat.openai.com/*`
- `https://claude.ai/*`

This access is used only to apply Temporary Chat behavior on ChatGPT pages and Incognito chat behavior on Claude pages.

## Remote Code

The extension does not load or execute remotely hosted code.

## Third-Party Services

The extension does not integrate with third-party analytics, advertising, tracking, or telemetry services.

## Contact

For privacy questions, open an issue in the GitHub repository for this extension.

## Affiliation

This extension is not affiliated with OpenAI or Anthropic.
